# 数据集格式转换
本仓库所涉及的多数数据集属于计算机视觉领域，随着仓库的不断扩充，后续会逐步推出诸如自然语言处理等领域的数据集格式转换脚本。

当前支持的数据集有VOC、COCO和YOLO。

如果你的数据集是自定义的，我十分推荐您将标注格式设置为VOC或COCO格式。
以便您能更好的使用不同的顶尖算法。
